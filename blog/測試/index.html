Markdown語法介紹

Markdown 是一種輕量級標記語言，目的是讓文檔編寫更簡單直觀。
它使用簡單的符號來標記文字的格式，並且可以輕鬆轉換為 HTML 格式。
以下是一些常用的 Markdown 語法及示例，幫助你快速學習並運用它來撰寫文檔。

1.標題（Headings）

Markdown 使用 # 標記來創建標題，# 的數量表示標題的層級。
最多支持 6 層標題（# 到 ######）。

---範例如下---
# 這是 H1 標題
## 這是 H2 標題
### 這是 H3 標題
#### 這是 H4 標題
##### 這是 H5 標題
###### 這是 H6 標題

2.段落（Paragraphs）

段落直接輸入文本即可，若需要換行，請按兩次 Enter。單行換行使用 （兩個空格）結尾。

---範例如下---
這是一個段落，按 Enter 會換行。

這是第二個段落。
