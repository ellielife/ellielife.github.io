HTML & Markdown 內文標簽的用法
HTML Markdown
<p>段落<\p> 無須寫程式碼，直接寫上文字即可，末尾須加兩個空格。
<br/> 換行 直接使用空行來換行。
<b>我是粗體字</b> 或是 <strong>我是粗體字</strong> ** 我是粗體字 ** 或是 __ 我是粗體字 __
<i>我是斜體字</i> 或是 <em>我是斜體</em> * 我是斜體字* 或是 _我是斜體字_
<h5> </h5> *我是斜粗體*
<h6> </h6> ######

※補充:HTML的<b>和<strong>兩組標籤所呈現的結果都是粗體，都是相同的，但在使用時機上是有差別的。

  • <b>:僅將文字標記為粗體。
  • <storng>:除了文字標記為粗體外還具有被強調的含意。

在SEO的部分，搜尋引擎會給予<storng>較高的權重，<b>只是普通的文字。另外，在螢幕輔助閱讀中，<storng>可被讀取，<b>則不會被讀取。

哈囉你好嗎
