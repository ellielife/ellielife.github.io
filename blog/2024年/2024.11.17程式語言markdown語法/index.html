Markdown語法介紹

Markdown 是一種輕量級標記語言，目的是讓文檔編寫更簡單直觀。
它使用簡單的符號來標記文字的格式，並且可以輕鬆轉換為 HTML 格式。
以下是一些常用的 Markdown 語法及示例，幫助你快速學習並運用它來撰寫文檔。

1.標題

Markdown 使用 # 標記來創建標題，# 的數量表示標題的層級。
最多支持 6 層標題（# 到 ######）。

---範例如下---
# 這是 H1 標題
## 這是 H2 標題
### 這是 H3 標題
#### 這是 H4 標題
##### 這是 H5 標題
###### 這是 H6 標題

顯示結果如下

這是 H1 標題

這是 H2 標題

這是 H3 標題

這是 H4 標題

這是 H5 標題
這是 H6 標題

2.段落與換行

段落直接輸入文本即可，若需要換行，請按兩次 Enter。單行換行使用 （兩個空格）結尾。

---範例如下---
這是一個段落，按 Enter 會換行。

這是第二個段落。

3.文字格式 (Text Formatting)

支持 粗體、斜體、粗斜體、刪除線。

---範例如下---
**粗體文字**
*斜體文字*
***粗斜體文字***
~~刪除線文字~~

顯示結果如下

粗體文字
斜體文字
粗斜體文字
刪除線文字

4.引用

使用 > 來創建引用。引用可以嵌套或包含其他元素。

---範例如下---
> 這是一段引用。
> 
> - 引用內的列表項目
> - 引用內的另一個項目

5.列表

  • 無序列表 使用 - 、 * 或 +。
  • 有序列表 使用數字。
---範例如下---
- 項目 1
- 項目 2
  - 子項目

1. 項目一
2. 項目二
   1. 子項目一
   2. 子項目二

6.程式碼 (Code)

行內程式碼 使用 `。
區塊程式碼 使用三個反引號 ``` 或縮排四個空格。
語法高亮 使用語言標註。

---範例如下---
行內程式碼：`print("Hello World")`

```python
# 這是一個 Python 程式碼區塊
def hello():
    print("Hello World")

7.分隔線 (Horizontal Rule)

使用三個或更多 -、* 或 _。

---範例如下---
1. ---
2. ___
3. ***
以上三種寫法都可以。

8.連結與圖片 (Links and Images)

連結 使用 [文字](URL)。
圖片 使用 ![文字](圖片URL)。
參考式連結 可重複使用同一連結。

---範例如下---
連結→ [Google](https://www.google.com)

圖片→ ![GitHub Logo](https://github.githubassets.com/images/modules/logos_page/Octocat.png)

[GitHub]: https://github.com
[這是 GitHub 的連結][GitHub]

9.內嵌html語法

Markdown 支援 HTML 語法，適用於標準 Markdown 無法處理的情況。

---範例如下---
<p style="color:red;">這是一段紅色文字。</p>

10.任務列表 (Task List)

使用 [ ] 表示未完成，[x] 表示已完成。

---範例如下---
- [x] 完成項目
- [ ] 未完成項目

顯示結果如下

  • 完成項目
  • 未完成項目

11.表格 (Tables)

使用 | 和 - 來創建表格，可以對齊內容。

---範例如下---
| 姓名  | 年齡 | 性別 |
|:-----|:----:|-----:|
| 小明  | 20   | 男   |
| 小花  | 18   | 女   |

顯示結果如下

姓名 年齡 性別
小明 20 男
小花 18 女

擴展語法 (Extended Syntax)

一些平台如 GitHub、VS Code 支援擴展語法。

1.折疊內容 (Details Tag)

<details>
<summary>點擊展開詳細內容</summary>
這是一段隱藏的詳細內容。
</details>

2.表情符號 (Emojis) 使用 : 包裹表情符號名稱，僅限支援的平台。

---範例如下---
:smile: :heart: :+1:

顯示效果：😄 ❤️ 👍

3.頁內連結 Markdown 中，標題通常會自動生成對應的錨點（Anchor）。可以使用 連結文字 的格式來建立頁內連結。
