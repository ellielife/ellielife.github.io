Markdown語法

By 喜歡閱讀的Ellie✨ 2024-09-18 前端標記語言

Markdown語法簡介

Markdown是由Daring Fireball創建的，原始指南在這裡。然而，它的語法在不同的解析器或編輯器之間有所不同。

請注意，Markdown 來源中的 HTML 片段將被識別，但不會被解析或渲染。另外，儲存後可能會對原始 Markdown 原始碼進行小的重新格式化。

內嵌 HTML

Markdown語法，用來作為網路寫作的格式。它不是HTML的替代品。它的語法非常小，僅對於HTML標籤是一個非常小的子集，Markdown想法是讓閱讀、寫作和編輯散文變得容易。

因此，HTML是一種發佈格式； Markdown 是一種寫作 格式。因此，Markdown 的格式化語法僅解決可以用純文字傳達的問題。

對於 Markdown 語法未涵蓋的任何標記，您只需使用 HTML 本身即可。無需在其前面加上前綴或對其進行分隔來表明您要從 Markdown 切換到 HTML；你只需使用標籤。

標籤 Heading

以#為符號代表。

程式碼範例如下…
#  this is an h1   
##   this is an h2 
###   this is an h3 
####   this is an h4 
#####   this is an h5  
######   this is an h6
顯示出來的結果…

this is an h1

this is an h2

this is an h3

this is an h4

this is an h5
this is an h6

清單

有序清單 List

以1.數字符號代表， 輸入1. list item 1將建立一個有序列表。

程式碼範例如下…
1. 黃色
2. 綠色
3. 藍色
顯示結果如下…
  1. 黃色
  2. 綠色
  3. 藍色

無序清單 Unordered List

以*和+或-為符號代表， 輸入* list item 1將建立一個無序列表。

程式碼範例如下…
* 黃色
* 綠色
* 藍色
顯示結果如下…
  • 黃色
  • 綠色
  • 藍色

任務清單

任務清單是其中項目標記為[ ]或[x]（不完整或完整）的清單。

程式碼範例如下…
- [] 12點睡覺
- [X] 鬧鐘設定好了嗎
顯示結果如下…
  • [] 睡覺
  • 鬧鐘設定好了嗎
