Boostrap5 導覽列基本觀念和類型

By 喜歡閱讀的Ellie✨ 2024-09-12 Boostrap5 rwd框架

Boostrap5 導覽列 基本觀念

navbar-expand-{-sm|-md|-lg|-xl|-xxl}，用來設定響應式折疊。
navbar-dark和bg-dark 可以設定顏色主題
在 Bootstrap 5 中，導覽列（Navbar）有多種類型，可以根據需求進行自訂和設置。以下是常見的導覽列類型和範例，涵蓋基礎的導航欄設置、帶有下拉菜單的導覽列、不同的定位方式、垂直導覽列等。

1. 基礎導覽列

這是一個簡單的 Bootstrap 5 導覽列範例，包含品牌名稱和導航連結。

範例圖片→ 

<body>
  <nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
      <a class="navbar-brand" href="#">品牌名稱</a>
      <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="切換導航">
        <span class="navbar-toggler-icon"></span>
      </button>
      <div class="collapse navbar-collapse" id="navbarNav">
        <ul class="navbar-nav">
          <li class="nav-item">
            <a class="nav-link active" aria-current="page" href="#">首頁</a>
          </li>
          <li class="nav-item">
            <a class="nav-link" href="#">功能</a>
          </li>
          <li class="nav-item">
            <a class="nav-link" href="#">價格</a>
          </li>
          <li class="nav-item">
            <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">停用</a>
          </li>
        </ul>
      </div>
    </div>
  </nav>

</body>

2. 設定導覽列主題顏色

主題顏色可以參考Boostrap5官網

※可使用.navbar-light將導覽列設定淺色背景顏色。
※可使用.navbar-dark將導覽列設定深色背景顏色。
※可使用.bg-實用程式進行自訂。

※以下是範例圖片

※以下是範例程式碼

<nav class="navbar navbar-dark bg-dark">
  <!-- Navbar content -->
</nav>

<nav class="navbar navbar-dark bg-primary">
  <!-- Navbar content -->
</nav>

<nav class="navbar navbar-light" style="background-color: #e3f2fd;">
  <!-- Navbar content -->
</nav>

4. 固定在頁面頂部的導覽列

使用fixed-top類別可以將導覽列固定在頁面頂部。
※可以套用在container語法裡，不一定要套用在.navbar上。

<nav class="navbar fixed-top navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Fixed top</a>
  </div>
</nav>

5. 固定在頁面底部的導覽列

使用fixed-bottom類別可以將導覽列固定在頁面頂部。
※可以套用在container語法裡，不一定要套用在.navbar上。

<nav class="navbar fixed-bottom navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Fixed bottom</a>
  </div>
</nav>

6. 浮動的導覽列

功能跟固定在頁面頂部一樣，不一樣的地方是若前面已先用第一個.container包裹起來的，那麼它的效果僅會浮動在第一個.container內，並不會隨著滑鼠滾輪往下後繼續浮動，要注意這一點。

使用sticky-top類別可以將導覽列固定在頁面頂部。
※可以套用在container語法裡，不一定要套用在.navbar上。
※另請注意，.sticky-top使用position: sticky，並非每個瀏覽器都完全支援。

<nav class="navbar sticky-top navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Sticky top</a>
  </div>
</nav>
